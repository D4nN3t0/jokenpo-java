## 🎮 Jokenpô em Java

Um jogo clássico de **Pedra, Papel e Tesoura** desenvolvido em Java, executado diretamente no terminal.

## 🧠 Funcionalidades

* 🎯 Jogador vs Máquina
* 🤖 Escolha aleatória do computador
* 💬 Mensagens dinâmicas para vitória, derrota e empate
* 🏆 Sistema de pontuação
* 📊 Exibição de placar durante o jogo
* 🔁 Modo **Melhor de 3**
* 🖥️ Interface simples via console

## 🚀 Como executar

```bash
javac Main.java
java Main
```

---

## 🎮 Como jogar

Escolha uma das opções no menu:

```
[0] Sair
[1] Pedra
[2] Papel
[3] Tesoura
[4] Mostrar pontuação
```

O jogo termina quando:

* Um dos jogadores vence 2 rodadas
* Ou após 3 partidas

## 📚 Tecnologias utilizadas

* ☕ Java
* 🧾 Programação Orientada a Objetos (POO)
* 📁 Estruturas básicas (arrays, condicionais e loops)
